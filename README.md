Overview
The Student Pay & Expense Tracker is a simple web application designed to help students manage their finances more easily. It brings together work‑hour logging, automatic earnings calculation, expense tracking, rent deductions, and balance updates — all in one clean dashboard. The goal is to give students a quick, accurate view of their weekly and monthly financial situation without needing spreadsheets or manual calculations.

Problem
Students often work part‑time jobs while studying, and it becomes difficult to keep track of:
How many hours they’ve worked
How much they’ve earned
How much they’ve spent
When rent is due
What their real remaining balance is
Most tools are either too complicated or don’t combine all these features in one place. This leads to confusion, miscalculations, and poor financial visibility.

Solution
This project provides a simple, student‑friendly financial tracker that allows users to:
Log work hours with date and description
Automatically calculate earnings based on hourly pay
View weekly summaries (hours worked, hours remaining, weekly earnings)
Add expenses by category
Apply monthly rent deductions
Receive last‑month pay with one click
See their balance update instantly
View logs of all work entries and expenses
The interface is designed to be minimal, fast, and easy to use daily.

Tech Stack

Backend: FastAPI (Python)
Database: SQLite
Server: Uvicorn
